# Mobile Sprint
## Overview
Este é um aplicativo de demonstração que permite visualizar clientes e os produtos que adquiriram, facilitando análises de vendas.

### Tecnologias Utilizadas
- Linguagem: Kotlin
- Framework: Android SDK
- Build Tool: Gradle
